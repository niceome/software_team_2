# 서비스 : TableMate


프로젝트 소개

대학생들은 종종 다음과 같은 상황을 경험합니다.
	•	혼자 점심을 먹기 애매할 때
	•	새로운 사람들과 교류하고 싶을 때
	•	같은 학교 학생들과 식사 약속을 잡고 싶을 때

TableMate는 이러한 문제를 해결하기 위해 만들어진 서비스입니다.

사용자는 점심 식사 모집 글을 작성하거나 다른 학생이 올린 글에 참여하여 함께 식사할 수 있습니다.


 주요 기능
	•	점심 식사 모집 게시글 작성
	•	함께 식사할 사람 모집
	•	댓글을 통한 참여 및 소통
	•	대학 기반 커뮤니티
	•	모집 알림 및 참여 관리
